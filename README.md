# MLOPs_AI
MLOPs ACP AI

## Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
   ```sh
   git clone https://github.com/aimarker/MLOPs_AI.git
   ```
3. Setup environments
   ```sh
   make setup
   ```
4. Install required packages
   ```sh
   make install
   ```
5. Check code styles
   ```sh
   make lint
   ```
6. Run tests
   ```sh
   make test
   ```
